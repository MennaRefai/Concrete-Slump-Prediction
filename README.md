# Concrete-Slump-Prediction
Built a regression model to predict concrete slump based on mixture components (cement, water, aggregates, superplasticizer). Performed EDA, feature scaling, Linear Regression with CV and regularization (Ridge, Lasso, Elastic Net). Evaluated using RMSE, MAE, R²
